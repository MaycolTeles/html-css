Trabalhando com Listas

Listas Ordenadas

Para criar listas ordenadas, usaremos a tag <ol>. Essa tag significa uma lista ordenada (ordered list, em inglês). Para criar cada elemento da lista, usaremos a tag <li>, representando um item da lista (list item, em inglês). Caso queira criar vários "li" de uma vez, basta digitar li*numero_elementos, onde numero_elementos é o número de li's que você deseja criar (exemplo: digite "li*7" e dê enter. isso irá criar 7 "li's".

  1. Acordar
  2. Tomar café
  3. Escovar os dentes

Para alterar o tipo do índex da lista, basta adicionar o parâmetro "type" da tag <ol>. Esse parâmetro aceita os possíveis valores: "1", "A", "a", "I", "i".

Listas Não Ordenadas

São listas onde a sequência não importa. É apenas para listar, independente da numeração. Para criá-las, usaremos a tag <ul> de unordered list (lista não ordenada, em inglês).

  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Carne

Para alterar o tipo do índex da lista, basta adicionar o parâmetro "type" da tag <ul>. Esse parâmetro aceita os possíveis valores: "disc", "circle" e "square".

Lista de Definições

Existe também a lista de definições (definition list, em inglês). Para declará-las, usaremos a tag <dl>. Seus elementos são compostos por um termo (cuja tag é <dt> e uma descrição (cuja tag é <dd>).

Termo
Descrição

Exemplo:

HTML
Linguagem de marcação para a criação do conteúdo de um site.
CSS
Linguagem de marcação para a criação do design de um site.
JavaScript
Linguagem de programação para a criação de interatividade de um site (Frontend)
Python
Linguagem de programação para a criação do processamento da parte do servidor de um site (Backend)

Listas de definições são usadas por buscadores (como o Google) que dá valor para sites que tem listas de definições em seu interior.
