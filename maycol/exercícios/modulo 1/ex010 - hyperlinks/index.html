Quando o link for direcionado para um site externo, abriremos ele em uma nova guia. Caso o link seja direcionado para um site interno (nosso site) iremos abrir na mesma guia.

Usando links externos

Você pode acessar meu repositório público no GitHub aqui.

Para poder controlar onde o site de destino vai abrir, podemos usar o atributo target. Basta definir 'target="_blank"' que irá abrir em uma nova guia. Para indicar qual é a natureza do destino, podemos usar o atributo rel. Basta definir 'rel="external"' quando o link for para um site terceiro.
